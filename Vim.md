### 尾行模式

| 快捷键           | 介绍                                                     |
| ---------------- | -------------------------------------------------------- |
| `Ctrl`+`alt`+`V` & `Ctrl` + `;`           | `IdeaVim`与`idea`普通模式切换                     |
| `:`+`行号`           | 行号定位              |
###### 进入insert模式
| `i`          | insert模式              |
| `o`          | 在当前行下方插入新行并自动缩进              |


###### insert模式（只用于输入文字）
| 快捷键           | 介绍                                                     |
| ---------------- | -------------------------------------------------------- |
| `Ctrl` + `h`          | 删除光标前一个`字符`              |
| `Ctrl` + `w`          | 删除光标前一个`单词`              |
| `Ctrl` + `u`          | 删除光标至当前`行首`              |
| `Ctrl` + `o`          | 进入插入模式下的普通模式            |
